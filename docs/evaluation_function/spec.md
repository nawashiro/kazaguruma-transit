# 会話の評価関数

## 概要

現在は単なる加算減算で投稿を評価し、ランキング表示しています。これを pol.is 式のコンセンサス検出アルゴリズムに変更します。これにより、少数派が無視されにくくなり、包摂性に貢献できます。

## UI 要件

以下のような項目を表示します。DaisyUI の join でグループを切り替えることができます。ランキング的な数字は表示しません。

- グループ考慮型コンセンサス
- グループ A の賛成タイプの代表的意見
- グループ B の賛成タイプの代表的意見
- グループ C の賛成タイプの代表的意見
  ...

サンプルの Python コードには他にも様々なメソッドが含まれますが、この表示に不要なメソッドは実装しないでください。無駄は省きます。

## 技術

[polis-consensus-algorithm](polis-consensus-algorithm.md)では、Python コードによりアルゴリズムを解説しています。

実際には TypeScript で実装する必要があります。[ai-on-browther](ai-on-browther.md)を使用します。
