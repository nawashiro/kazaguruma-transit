## その他の最適化

### 電話番号自動検出の無効化

モバイルデバイスでの意図しない電話番号リンク化を防止しています。

```html
<meta name="format-detection" content="telephone=no" />
```

これは時刻表や路線番号などの数字が電話番号として誤認識されることを防ぎます。

### プリフェッチの設定

ユーザーがよく訪問するページへのプリフェッチを設定し、ページ遷移の高速化を図っています。

```html
<link rel="prefetch" href="/beginners-guide" />
<link rel="prefetch" href="/usage" />
```
