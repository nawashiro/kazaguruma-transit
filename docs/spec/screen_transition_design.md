# 画面遷移構成

## 画面一覧

- ホーム: 乗換案内を提供します。
- 場所をさがす: 風ぐるまで到達可能なスポットの一覧を提供します。
- はじめての方へ: 初めてアクセスした人向けの案内です。静的ページです。
- 使いかた: マニュアルです。静的ページです。
- ライセンス: データやライブラリのライセンス情報を記載します。静的ページです。
- 更新情報: 外部サイトです。Notion により提供されています。
- サイトマップ: 各種ページの一覧です。静的ページです。

## 遷移

- ほとんどのページはナビゲーションにより互いに遷移することができます。
- 場所をさがす → ホーム: 場所を選択し、ホームに遷移します。選択した場所が既に入力された状態になります。
- サイトマップ: すべてのページへ遷移します。
