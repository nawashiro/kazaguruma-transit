# Data Model: 会話タブナビゲーション修正

## Entities

### 会話

- **説明**: 個別の会話単位
- **主要属性**: 既存の識別子、作成者、承認状態
- **関係**: 作成者/モデレーターの権限判定に利用される

### ユーザー権限

- **説明**: 作成者・モデレーター・一般ユーザーの区分
- **主要属性**: 既存の権限フラグや役割情報
- **関係**: 会話に対する操作可否と表示内容に影響

## Validation Rules

- 権限判定は既存のルールを踏襲し、追加の制約は設けない

## State Transitions

- 本機能による状態遷移の追加はなし
