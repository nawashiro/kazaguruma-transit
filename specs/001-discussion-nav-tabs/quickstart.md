# Quickstart: 会話タブナビゲーション修正

## 目的

会話詳細ページのタブレイアウト変更、説明ブロックの常時表示、旧導線の撤去を確認する。

## 前提

- `npm install` 済み
- 必要な環境変数を設定済み

## 確認手順

1. `npm run dev` でアプリを起動する
2. 会話詳細ページを開き、タブレイアウト内の承認/編集リンクの表示条件と順序を確認する
3. タブレイアウト最下部の説明ブロックが常時表示され、役割に応じた文面が表示されることを確認する
4. 旧ブロック表示が存在しないことを確認する
5. `/discussions/[id]/edit` と `/discussions/[id]/approve` から「会話に戻る」が削除されていることを確認する

## テスト

- `npx tsc --noEmit`
- `npm run lint`
- `npm test`
- `npm run build`
