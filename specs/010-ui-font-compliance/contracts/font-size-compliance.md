# UI文字サイズ準拠契約

## 対象

アプリケーションが利用者へ表示する、ページ、ナビゲーション、フォーム、ボタン、リンク、
ダイアログ、メニュー、通知、空状態、読み込み状態、エラー状態のテキスト。

## 不変条件

1. ルビ補助テキスト以外の対象テキストは、表示フォントサイズが 14px 以上である。
2. 幅や状態が変わっても、不変条件 1 を破らない。
3. 14px 未満を許容するのは、本文に付随するルビ補助テキストだけである。
4. アイコンのみのUI、画像に焼き込まれた文字、第三者コンテンツ、PDF出力はこの契約の対象外である。

## 検証境界

- 自動検証は通常UIで使われる 14px 未満のサイズ指定を失敗として報告する。
- ルビ用 `rt` スタイルは例外として明示的に区別する。
- 最終確認は代表的な画面を広い・狭い表示幅で表示し、上記の不変条件を確認する。
